# humanizer

## Что делает

- Вырезает запретные слова: delve, tapestry, realm, landscape,
  navigate, leverage (в значении «использовать»), robust, seamless,
  game changer, revolutionary, unprecedented.
- Вырезает запретные фразы: «в мире криптовалют», «стоит отметить»,
  «не секрет что», «in the ever-evolving».
- Если в тексте больше одной конструкции «не X, а Y» —
  оставляет только первую, остальные упрощает.
- Заменяет em-dash `—` на дефис `-` или точку.
- Не требует заглавной буквы в начале и точки в конце
  (твиттерный стиль ок).
- Режет результат до **270 символов** с учётом ссылки.

## Запуск

```bash
python .claude/skills/humanizer/humanize.py
```

На вход берёт `workspace/tweet_raw.txt`, на выход пишет
`workspace/tweet_final.txt` и печатает в stdout.

## Правило

**Humanizer НИКОГДА не пропускается в пайплайне.**
Даже если твит уже выглядит норм — прогнать.
